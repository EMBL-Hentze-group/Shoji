# CLIP_savvy

A replacement for [htseq-clip](https://htseq-clip.readthedocs.io/en/latest/).

Major differences:  

* No `--splitExons` flag, savvy_clip cannot split exons into components  
* Piping output disabled. Output file names MUST be specified
